### 1. **Reading Input**

Java uses the `Scanner` class (from `java.util` package) for simple input operations.

#### Example: Reading Different Data Types

```java
import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Reading a string
        System.out.print("Enter your name: ");
        String name = scanner.nextLine();

        // Reading an integer
        System.out.print("Enter your age: ");
        int age = scanner.nextInt();

        // Reading a double
        System.out.print("Enter your salary: ");
        double salary = scanner.nextDouble();

        System.out.println("Hello, " + name + ". You are " + age + " years old and earn $" + salary);

        scanner.close(); // Close the scanner to release resources
    }
}
```

### 2. **Printing Output**

Java uses the `System.out` object for output. The common methods are:

#### a. `System.out.print()`

- Prints text **without** a newline.
- Example:
    
    ```java
    System.out.print("Hello ");
    System.out.print("World");
    // Output: Hello World
    ```
    

#### b. `System.out.println()`

- Prints text **with** a newline.
- Example:
    
    ```java
    System.out.println("Hello");
    System.out.println("World");
    // Output:
    // Hello
    // World
    ```
    

#### c. `System.out.printf()`

- Formats output using a format specifier.
- Example:
    
    ```java
    System.out.printf("Name: %s, Age: %d, Salary: %.2f\n", "Alice", 30, 4500.75);
    // Output: Name: Alice, Age: 30, Salary: 4500.75
    ```

### 3. **Using BufferedReader for Input (Alternative)**

For faster input, you can use `BufferedReader`.

#### Example:

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class BufferedReaderExample {
    public static void main(String[] args) throws IOException {
        BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));

        System.out.print("Enter your name: ");
        String name = reader.readLine();

        System.out.print("Enter your age: ");
        int age = Integer.parseInt(reader.readLine());

        System.out.println("Hello, " + name + ". You are " + age + " years old.");
    }
}
```

---

### 4. **Using Command-Line Arguments**

Java programs can also accept input directly via command-line arguments.

#### Example:

```java
public class CommandLineExample {
    public static void main(String[] args) {
        if (args.length == 2) {
            String name = args[0];
            int age = Integer.parseInt(args[1]);

            System.out.println("Hello, " + name + ". You are " + age + " years old.");
        } else {
            System.out.println("Please provide your name and age as arguments.");
        }
    }
}
```

---

### 5. **Error Stream (`System.err`)**

Used to print error messages.

```java
System.err.println("This is an error message.");
```

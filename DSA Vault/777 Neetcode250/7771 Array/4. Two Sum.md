# Two Sum
[[Array]]
Given an array of integers `nums` and an integer `target`, return the indices `i` and `j` such that `nums[i] + nums[j] == target` and `i != j`.

You may assume that _every_ input has exactly one pair of indices `i` and `j` that satisfy the condition.

Return the answer with the smaller index first.

**Example 1:**

```java
Input: 
nums = [3,4,5,6], target = 7

Output: [0,1]
```

Explanation: `nums[0] + nums[1] == 7`, so we return `[0, 1]`.

**Example 2:**

```java
Input: nums = [4,5,6], target = 10

Output: [0,2]
```

**Example 3:**

```java
Input: nums = [5,5], target = 10

Output: [0,1]
```

**Constraints:**

- `2 <= nums.length <= 1000`
- `-10,000,000 <= nums[i] <= 10,000,000`
- `-10,000,000 <= target <= 10,000,000`
- **Only one valid answer exists.**

**Solution:**
```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int[] result= new int[2];
        for(int i=0;i<nums.length;i++){
            for(int j=0;j<nums.length;j++){
                if((nums[i]+nums[j])==target && i !=j){
                    result[0]=i;
                    result[1]=j;
                }
            }
        }
        Arrays.sort(result);
        return result;
    }
}
```
Time complexity: O(n2)  
Space complexity: O(1)

`n` is the number of elements in the `nums` array.

**Online Solution:**
```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        int[] result= new int[2];
        for(int i=0;i<nums.length;i++){
            for(int j=i+1;j<nums.length;j++){
                int complement=target-nums[i];
                if(nums[j]==complement){
                   return new int[] {i, j};
                }
            }
        }
        throw new IllegalArgumentException("no match found");
    }
}
```

Optimized Solution:
```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer,Integer> nums_map=new HashMap<>();
        for(int i=0;i<nums.length;i++){
            int complement=target-nums[i];
            if(nums_map.containsKey(complement)){
                return new int[]{nums_map.get(complement),i};
            }
            nums_map.put(nums[i],i);
        }
        throw new IllegalArgumentException("no match found");
    }
}
```
